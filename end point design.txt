Nextagram API: 

GET: users_api/ - get all users information

GET: users_api/id - get specific user information

GET: user_api/images - Get request to retrieve all user images

GET: user_api/images/id - Get request to obtain all images for the user ID

GET: /me - Get request to attain information of user who currently log in


POST: /login - Post request to login to app

POST: /images/ - Post a image
